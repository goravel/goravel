# 关于 Goravel

Goravel 是一个功能完备、具有良好扩展能力的 Web 应用程序框架。 作为一个起始脚手架帮助 Golang 开发者快速构建自己的应用。

> 目前为开发版，新升级有可能未向前兼容，请慎重用于生产环境。

## 主要功能

- [x] 自定义配置
- [x] HTTP 服务
- [x] 数据库 ORM
- [x] 数据库迁移
- [x] 日志
- [x] 缓存
- [x] Grpc
- [x] 命令
- [x] 任务调度
- [ ] 队列
- [ ] 事件系统
- [ ] 邮件
